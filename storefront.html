STOREFRONT API v1

Temalar için tek, hızlı ve güvenli veri sözleşmesi.

Shopiyz temaları D1'e bağlanmaz. Aynı origin Storefront Runtime bu API'yi çağırır; API Worker tarafında mağazanın D1 ve R2 binding'leriyle veriyi hazırlar.

Aynı origin tema

Shopiyz temaları public GET uçlarını token taşımadan çağırır. Müşteri ve checkout işlemleri güvenli oturum çereziyle aynı origin'de kalır.

const page = await fetch(
  "/api/storefront/v1/page?pageType=home&path=/"
).then(r => r.json());

Headless istemci

Harici istemci mağazaya bağlı Storefront tokenı kullanır. Token yalnızca storefront:read kapsamına sahiptir.

curl -H   "X-Shopiyz-Storefront-Access-Token: shpft_…"   "https://store.myshopiyz.com/api/storefront/v1/products"

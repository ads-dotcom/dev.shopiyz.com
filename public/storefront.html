STOREFRONT API v1

Temalar için hızlı, güvenli ve tutarlı veri katmanı.

Shopiyz temaları D1'e bağlanmaz. Aynı-origin Storefront Runtime veya mağazaya bağlı headless token; fiyat, varyant, menü, filtre, içerik ve kampanya verisini tek sürümlü sözleşmeden alır.

Mağaza bazlı erişimHer istek doğru mağaza ve yayınlanmış tema bağlamında çözülür.
Güvenli kapsamHeadless token yalnızca storefront:read yetkisi taşır.
Edge performansıKatalog ve içerik yanıtları kararlı cache politikalarıyla sunulur.
BaşlayınEndpointleri inceleyin
JavaScript
// Ana sayfa tema verisini alın
const page = await fetch(
  "/api/storefront/v1/page?pageType=home&path=/"
).then((response) => response.json());

renderTheme(page);

QUICKSTART

Doğru istemci tipini seçerek başlayın

Aynı-origin temalar public uçları kullanır; harici istemciler mağazaya bağlı Storefront tokenı taşır.

Aynı-origin tema

Public GET uçları token taşımadan çağrılır. Müşteri ve checkout işlemleri güvenli oturum çereziyle aynı origin'de kalır.

const page = await fetch(
  "/api/storefront/v1/page?pageType=home&path=/"
).then((response) => response.json());

Headless istemci

Harici istemci yalnızca storefront:read kapsamına sahip mağaza tokenını kullanır.

curl -H   "X-Shopiyz-Storefront-Access-Token: shpft_…"   "https://store.myshopiyz.com/api/storefront/v1/products"

Koleksiyon filtreleri

Facet değerleri ve ürünler aynı yanıtta gelir; tema yalnız URL sorgusunu yönetir.

const listing = await fetch(
  "/api/storefront/v1/collections/all" +
  "?color=Siyah&size=M&sort=price-asc&page=1"
).then((response) => response.json());

Fiyat ve kampanya

Fiyatlar mağaza para birimiyle biçimlenir; yalnız herkese açık otomatik promosyonlar okunur.

const localization = await fetch(
  "/api/storefront/v1/localization"
).then((response) => response.json());

TEMA GELİŞTİRME KAPSAMI

Harici bir temanın ihtiyaç duyduğu veri yüzeyi

ThemeForest veya özel bir frontend, yönetim veritabanına bağlanmadan aşağıdaki sözleşmelerle çalışır.

Navigasyon

Header, mega menu, mobil menü ve footer için konumlu, çok seviyeli /menus ağacı.

Kategori ve filtre

Koleksiyon, marka, ürün tipi, kategori, etiket, renk, beden, stok ve fiyat facetleri; sayfalama ve sıralama.

Ürün ve fiyat

Varyant seçenekleri, görseller, stok, satış fiyatı, karşılaştırma fiyatı ve mağaza para birimi. Maliyet değeri maskelenir.

Kampanya

Aktif otomatik promosyonlar ve hedef ürün/koleksiyonlar. Özel müşteri kuralları ile indirim kodları hiçbir zaman açılmaz.

İçerik ve SEO

Sayfalar, bloglar, blog yazıları, SEO alanları, tema layout'u ve header/footer shell verisi.

Müşteri ve ticaret

Aynı-origin güvenli oturumla giriş, kayıt, parola, wishlist, yorum, stok bildirimi ve checkout devri.
